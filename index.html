⚙️ Nastavenia ukladania dát

Personal Access Token (PAT) s oprávnením contents:write na váš repozitár.

Uloží sa len do localStorage tohto prehliadača.

Prihlásenie cez Google OAuth 2.0. Súbor knjazd_data.json sa uloží do vášho Google Drive.

Získate v Google Cloud Console → APIs & Services → Credentials → OAuth 2.0 Client ID (Web app).
🔧 Nastavenie Google Cloud
  1. Otvorte console.cloud.google.com
  2. Vytvorte projekt → APIs & Services → Enable APIs → zapnite Google Drive API
  3. OAuth consent screen → External → zadajte názov app, email
  4. Credentials → Create → OAuth 2.0 Client ID → Web application
  5. Pridajte Authorized JavaScript origins: vašu GitHub Pages URL + http://localhost:8765
  6. Skopírujte Client ID a vložte ho sem

Dáta sú uložené len v tomto prehliadači. Funguje offline, ale nie je synchronizácia medzi zariadeniami.

  • Vymazanie histórie prehliadača môže zmazať dáta
  • Pravidelne exportujte zálohu cez Nastavenia → Export JSON
ℹ️ Priorita ukladania
  • localhost → Python server (data.json na disku)
  • Google Drive → ak je používateľ prihlásený
  • GitHub → ak je nakonfigurovaný PAT
  • localStorage → vždy ako záloha/fallback
