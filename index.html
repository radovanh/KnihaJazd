🐙 GitHub API (pre GitHub Pages hosting)

Potrebujete Personal Access Token (PAT) s oprávnením contents:write.

Uloží sa len do localStorage tohto prehliadača.
ℹ️ Ako to funguje
  • localhost → Python server → ukladá do data.json na disku + localStorage záloha
  • GitHub Pages + PAT → GitHub API → commit priamo do repozitára
  • GitHub Pages bez PAT → len localStorage (len tento prehliadač)
  • Export JSON je vždy k dispozícii ako manuálna záloha
